# Scroll-Driven Hero Animation

A scroll-driven hero section built using React, GSAP, and Tailwind.

## Features
- Smooth scroll-based animation using GSAP ScrollTrigger
- Staggered text and stats animations
- Parallax motion and easing
- Responsive design

## Tech Stack
- React (Vite)
- GSAP
- Tailwind CSS
